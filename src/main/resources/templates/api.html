Redis - база данных для хранения данных в формате ключ-значение.

Ключ - строка. Значением может быть строка, список и другие дата-структуры.

Control Api

Http method Endpoint Description
POST /api/run/{task_id} Run task

Data Api

Http method Endpoint Description

                            FormData

                            redisId: {redisId}
